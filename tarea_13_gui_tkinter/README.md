
# Aplicación GUI Básica con Tkinter

## Descripción
Esta aplicación fue desarrollada en Python utilizando la librería Tkinter.
Permite ingresar información mediante un campo de texto y mostrarla en una lista.

## Funcionalidades
- Agregar información mediante un campo de texto.
- Mostrar los datos en una lista dentro de la interfaz gráfica.
- Eliminar elementos seleccionados.
- Limpiar el campo de texto.

## Componentes GUI utilizados
- Label (etiquetas)
- Entry (campo de texto)
- Button (botones)
- Listbox (lista de datos)

## Requisitos
Tener instalado Python 3.

## Cómo ejecutar el programa
1. Abrir la terminal o consola.
2. Navegar hasta la carpeta del proyecto.
3. Ejecutar:

python main.py
